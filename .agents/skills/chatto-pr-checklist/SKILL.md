# Chatto PR Checklist

- Familiarize yourself with the changes in this branch/PR.
- Are there any test gaps around the new/changed functionality?
- Please check if our code documentation -- specifically ADRs, FDRs, glossary, and architecture inventory -- is up to date with the changes on this branch. Use the relevant skills to audit the documentation in parallel.
- Please also check if the docs-website needs an update.

For each of the above documentation, please present the user with a list of recommended actions. Only make changes with user approval — this skill is for auditing, not auto-fixing.
